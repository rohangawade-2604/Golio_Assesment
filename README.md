# Golio UI – React & Bootstrap Responsive Website

A modern, responsive user interface built using **React.js** and **Bootstrap 5**, focusing on clean layout, mobile-first design, and reusable components.

---

## Overview

Golio UI is a frontend project that demonstrates best practices in building responsive layouts using the Bootstrap grid system combined with React component architecture.  
The project includes a custom responsive navbar, cards, testimonials, and footer sections.

---

## Features

- Responsive layout (mobile, tablet, desktop)
- Custom hamburger menu for small screens
- Navigation links with social media icons
- Reusable React components
- Bootstrap 5 grid and utility classes
- Font Awesome icons
- Clean and maintainable code structure

---

## Tech Stack

- React.js
- Bootstrap 5
- CSS3
- JavaScript (ES6+)
- Font Awesome

## Responsive Behavior

| Breakpoint | Layout Behavior |
|-----------|----------------|
| < 576px | Hamburger menu, stacked content |
| ≥ 768px | Two-column layout |
| ≥ 992px | Full navigation with social icons |
